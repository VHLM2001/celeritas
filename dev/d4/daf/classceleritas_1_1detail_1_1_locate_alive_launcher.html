Detailed Description

template<MemSpace M>
class celeritas::detail::LocateAliveLauncher< M >

Determine which tracks are alive and count secondaries.

This finds empty slots in the track vector and counts the number of secondaries created in each interaction. If the track was killed and produced secondaries, the empty track slot is filled with the first secondary.
