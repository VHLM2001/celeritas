Detailed Description

Construct and store data for Urban multiple scattering.

Multiple scattering is used by the along-step kernel(s).

TODO: UrbanMsc is the only MSC model presently in Celeritas, but we should extend this to be an interface since it's an interchangeable component of the along-step kernel(s).
