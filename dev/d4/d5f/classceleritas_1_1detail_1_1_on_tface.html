Detailed Description

template<class ValueT>
class celeritas::detail::OnTface< ValueT >

Signed surface.

Particles are never allowed to be logically "on" a surface: they must be logically on one side or another so that they are in a particular volume.

This class encapsulates the storage of the sense/value: we could try using a bit field to compress the ID and sense at the cost of reducing the max number of surfaces/faces by a factor of two.
