Detailed Description

Axis-aligned bounding box.

This is currently a pretty boring class; it will be extended as needed. We'll add a BoundingBoxUtils class for all the fancy operations that need only the accessors we provide.

The bounding box can be constructed in an unassigned state, in which lower and upper cannot be called.
