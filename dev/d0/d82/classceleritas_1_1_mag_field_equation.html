Detailed Description

template<class FieldT>
class celeritas::MagFieldEquation< FieldT >

Evaluate the force applied by a magnetic field.

The templated FieldT must be a function-like object with the signature

Real3 (*)(const Real3&)
celeritas::Array< real_type, 3 >

which returns a magnetic field vector at a given position. The field strength is in Celeritas native units, so multiply by units::tesla if necessary.
