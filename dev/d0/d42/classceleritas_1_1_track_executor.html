Detailed Description

template<class T>
class celeritas::TrackExecutor< T >

Call a CoreTrackView executor for a given ThreadId.

This class can be used to call a functor that applies to CoreTrackView using a ThreadId, so that the tracks can be easily looped over as a group on CPU or GPU.

This is primarily used by ActionLauncher .
