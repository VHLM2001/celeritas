Detailed Description

Return a simplified version of a surface, possibly flipping associated sense.

This class is meant for using SurfaceAction or std::variant to visit a surface type.

The result of each simplification type should be a std::variant of possible simplified class forms, or a std::monostate if no simplification was applied.

The embedded sense may be flipped as part of the simplifications.

◆ operator()() [4/5]

auto celeritas::SurfaceSimplifier::operator() ( Plane const &  p)

Plane may be flipped, adjusted, or become axis-aligned.

If a plane has a normal of {-1, 0 + eps, 0}, it will first be truncated to {-1, 0, 0}, then flipped to {1, 0, 0}, and a new Plane will be returned. That plane can then be simplified to an axis-aligned one.
