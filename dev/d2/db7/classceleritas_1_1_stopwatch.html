Detailed Description

Simple timer.

The stopwatch starts counting upward at construction and can be reset by assigning a new stopwatch instance.

Stopwatch get_elapsed_time;
// ...
double time = get_elapsed_time();
// Reset the stopwatch
get_elapsed_time = {};
