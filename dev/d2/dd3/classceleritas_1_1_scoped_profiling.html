Detailed Description

RAII class for scoped profiling.

Implementations should support multithreaded context where each thread have one or more alive instance of this class.

This is useful for wrapping specific code fragment in a range for profiling, e.g. ignoring of VecGeom instantiation kernels, profiling a specific action or loop on the CPU. TODO: Template ScopedProfiling over profiling backend if we need to add a new one
