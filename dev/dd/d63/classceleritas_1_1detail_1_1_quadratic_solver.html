Detailed Description

Find positive, real, nonzero roots for quadratic functions.

The quadratic equation

\[ a x^2 + b^2 + c = 0 \]

has two solutions mathematically, but we only want solutions where x is real and positive. Furthermore the equation is subject to catastrophic roundoff due to floating point precision (see Tolerance::sqrt_quadratic and the derivation in CylAligned ).

Returns
An Intersections array where each item is a positive valid intersection or the sentinel result no_intersection() .

Solve degenerate case (known to be "on" surface).

Since x = 0 is a root, then c = 0 and x = -b is the other root. This will be inaccurate if a particle is logically on a surface but not physically on it.

CELER_FUNCTION auto celeritas::detail::QuadraticSolver::operator() ( real_type c) const
inline

Find all positive roots of x^2 + (b/a)*x + (c/a) = 0.

In this case, the quadratic formula can be written as:

\[ x = -b/2 \pm \sqrt{(b/2)^2 - c}. \]

Callers:

  • General quadratic solve: not on nor along surface
  • Sphere when not on surface
  • Cylinder when not on surface

inlinestatic

Solve degenerate case when parallel to but not on surface.

This is the case when a is approximately zero.

Note that as both a and b -> 0, |x| -> infinity. Thus, if b/a is sufficiently small, no positive root is returned (because this case corresponds to a ray crossing a surface at an extreme distance).
