Detailed Description

Find positive, real, nonzero roots for quadratic functions.

These are for quadratic functions

where a is nonzero (and not close to zero).

This is used for all quadrics with potentially two roots (anything but planes).

Each item in the Intersections result will be a positive valid intersection or the sentinel result no_intersection() .

Solve degenerate case (known to be "on" surface).

Since x = 0 is a root, then c = 0 and x = -b is the other root. This will be inaccurate if a particle is logically on a surface but not physically on it.
