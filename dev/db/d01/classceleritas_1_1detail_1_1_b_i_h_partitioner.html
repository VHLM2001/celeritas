Detailed Description

Partition bounding boxes using a surface area heuristic.

The class take a vector of bounding boxes as an input, and outputs a Partition object describing the optional partition. To find the optimal partition, all possible candidate partitions along the x, y, and z axis are evaluated using a cost function. The cost function is based on a standard surface area heuristic.
