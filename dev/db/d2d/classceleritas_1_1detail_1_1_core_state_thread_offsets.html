Detailed Description

template<MemSpace M>
class celeritas::detail::CoreStateThreadOffsets< M >

Holds Collections used by CoreState to store thread offsets.

This is specialized for device memory space as two collections are needed, one for the host and one for the device. Using pinned mapped memory would be less efficient.
