Detailed Description

Apply a translation to a transform to get another transform.

An instance of this class applies a daughter-to-parent translation to another transformation: the translation needed to relocate a "lower" universe to a new coordinate system in the "higher" universe.

Given a transform operator T defined

\[ \mathbf{T} x = \mathbf{R} x + t \]

and a translation operator defined

\[ \mathbf{\tilde T} x = x + t \]

where \( \mathbf{R} = \mathbf{I} \) is the implicit rotation matrix for a translation, this operation returns a new transform

\[ \mathbf{T}' = \mathbf{\tilde T}_A \mathbf{T}_B \]

where \(\mathbf{\tilde T}_A\) is the constructor argument, T_B is the argument passed to the call operator, and the \(\mathbf{T}'\) is the return value. The resulting transform has rotation

\[ \mathbf{R}' = \mathbf{R}_B \]

and translation

\[ \mathbf{t}' = \mathbf{t}_B + \mathbf{t}_A \]
