Detailed Description

template<Ownership W, MemSpace M>
struct celeritas::SurfaceData< W, M >

Data for type-deleted surface definitions.

Surfaces each have a compile-time number of real data needed to define them. (These usually are the nonzero coefficients of the quadric equation.) A surface ID points to an offset into the data field. These surface IDs are global over all universes.
