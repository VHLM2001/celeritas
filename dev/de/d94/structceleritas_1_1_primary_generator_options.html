Detailed Description

Primary generator options.

  • pdg: PDG numbers of the primaries. An equal number of primaries of each type will be generated
  • num_events: total number of events to generate
  • primaries_per_event: number of primaries to generate in each event
  • energy: energy distribution type and parameters
  • position: spatial distribution type and parameters
  • direction: angular distribution type and parameters
