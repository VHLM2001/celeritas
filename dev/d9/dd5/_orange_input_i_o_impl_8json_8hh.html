std::vector< logic_int > celeritas::detail::string_to_logic ( std::string const &  s)

Build a logic definition from a C string.

A valid string satisfies the regex "[0-9~!| ]+", but the result may not be a valid logic expression. (The volume inserter will ensure that the logic expression at least is consistent for a CSG region definition.)

Example:

parse_logic("4 ~ 5 & 6 &");
