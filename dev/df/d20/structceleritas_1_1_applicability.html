Detailed Description

Range where a model and/or process is valid.

This class is used during setup for specifying the ranges of applicability for a physics model or process. The interval is open on the lower energy range and closed on the upper energy. So a threshold reaction should have the lower energy set to the threshold. Models valid to zero energy but have special "at rest" models should set upper to zero.

An unset value for "material" means it applies to all materials; however, the particle ID should always be set.
