size_type celeritas::detail::exclusive_scan_counts ( StateCollection< size_type, Ownership::reference, MemSpace::host > const & counts,
StreamId  )

Do an exclusive scan of the number of secondaries produced by each track.

For an input array x, this calculates the exclusive prefix sum y of the array elements, i.e., \( y_i = \sum_{j=0}^{i-1} x_j \), where \( y_0 = 0 \), and stores the result in the input array.

The input size is one greater than the number of track slots so that the final element will be the total accumulated value.
