Detailed Description

Apply a transformation to another transformation.

An instance of this class applies a daughter-to-parent translation to another transformation: the translation needed to relocate a "lower" universe to a new coordinate system in the "higher" universe.

Given a transform operator T defined

\[ \mathbf{T}x = \mathbf{R} x + t \]

this operation returns a new transform

\[ \mathbf{T}' = \mathbf{T}_A \mathbf{T}_B \]

where \(\mathbf{T}_A\) is the constructor argument (the operator itself), T_B is the argument passed to the call operator, and the \(\mathbf{T}'\) is the return value. The new transform has rotation

\[ \mathbf{R}' = \mathbf{R}_A\mathbf{R}_B \]

and translation

\[ \mathbf{t}' = \mathbf{R}_A\mathbf{t}_B + \mathbf{t}_A \]
