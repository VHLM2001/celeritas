Detailed Description

RAII class for CUDA/HIP profiling flags .

This should be instantiated in a single-thread context. If celeritas::device is enabled, it will call cudaProfilerStart at construction and cudaProfilerStop at destruction. If device support is disabled, this class is a null-op. HIP support for the profiler start/stop macros is deprecated so we do not support it at present.

This is useful for wrapping the main stepper/transport loop for profiling to allow ignoring of VecGeom instantiation kernels.
