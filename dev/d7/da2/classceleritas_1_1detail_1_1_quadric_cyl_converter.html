Detailed Description

Try to convert a simple quadric to a cylinder.

The simple quadric must have already undergone normalization (one second-order term approximately zero, the others positive).

Try converting to a cylinder with this orientation.

Cone: 

   (x - x_0)^2 + (y - y_0)^2 - r^2 = 0
*

Expanded: 

          x^2 +      y^2
   - 2x_0 x   - 2y_0 y
   + (x_0^2 + y_0^2 - r^2) = 0
*

SQ: 

  a x^2 + (a + \epsilon) y^2 + e x + f y + h = 0
*

Normalized (epsilon = 0) 

  x^2 + y^2 + e/b x + f/b y + g/b z  + h/b = 0
*

Match terms: 

   -2 x_0 = e/b --> x_0 = e / (-2 * b)
   -2 y_0 = f/b --> y_0 = f / (-2 * b)
  (x_0^2 + y_0^2 - r^2) = h/b
     -> r^2 = x_0^2 + y_0^2 - h/b
*
