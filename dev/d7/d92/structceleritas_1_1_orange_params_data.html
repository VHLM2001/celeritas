Detailed Description

template<Ownership W, MemSpace M>
struct celeritas::OrangeParamsData< W, M >

Persistent data used by ORANGE implementation.

Most data will be accessed through the invidual units, which reference data in the "storage" below. The type and index for a universe ID will determine the class type and data of the Tracker to instantiate. If only simple units are present, then the simple_unit data structure will just be equal to a range (with the total number of universes present). Use universe_type to switch on the type of universe; then universe_index to index into simple_unit or rect_array or ...
