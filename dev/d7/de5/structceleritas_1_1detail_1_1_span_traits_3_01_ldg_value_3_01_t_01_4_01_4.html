Detailed Description

template<class T>
struct celeritas::detail::SpanTraits< LdgValue< T > >

Type aliases when data is read using __ldg.

LdgValue checks that T is a valid type (const arithmetic or OpaqueId). Since LdgIterator returns a copy of the data read, we can't return a reference to the original data, we need to return a copy as well.
