Detailed Description

Construct surfaces on the host.

Currently this simply appends the surface to the Data, but the full robust geometry implementation will implement "soft" surface deduplication.

auto id = insert_surface(PlaneX(123));
auto id2 = insert_surface(PlaneX(123.0000001)); // TODO: equals id
