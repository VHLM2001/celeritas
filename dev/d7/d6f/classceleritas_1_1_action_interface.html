Detailed Description

Pure abstract interface for an end-of-step action.

The action ID is used to select between post-step actions such as discrete processes, geometry boundary, and range limitation.

The ActionInterface provides a no-overhead virtual interface for gathering metadata (and someday for launching kernels).
