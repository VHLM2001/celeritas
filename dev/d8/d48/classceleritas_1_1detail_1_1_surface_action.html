Detailed Description

template<class F>
class celeritas::detail::SurfaceAction< F >

Helper class for applying an action functor to a generic surface.

The function-like instance of F must accept any surface type as an argument: this should always just be a templated operator() on the surface class. The result type should be the same regardless of the surface type.
