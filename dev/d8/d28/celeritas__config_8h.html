Detailed Description

Configuration-specific options for Celeritas.

Note that the nonzero values for CELERITAS_CORE_RNG and CELERITAS_CORE_GEO values must not be used directly: only compare between (e.g.) CELERITAS_CORE_RNG and CELERITAS_CORE_RNG_HIPRAND; options that are invalid (e.g. for missing libraries such as HIP) will have a value of

zero.
